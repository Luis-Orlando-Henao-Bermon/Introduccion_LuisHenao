## Carpetas y Contenidos

### **Dia_1**

|Programa|Funcion|
|--|--|
|Ejercicio1_HenaoLuis.psc|El programa es para sumar 2 numeros solicita ingresar dos números y como resultado da la suma de ambos.|
|Ejercicio2_HenaoLuis.psc|El programa es para ver el mayor de 3 numeros, solícita ingresar 3 números para posteriormente mostrar cuál es el número mayor de los 3.|
|Ejercicio3_HenaoLuis.psc|El programa es para ver el factorial de un numero, solícita un número y da como resultado el factorial del mismo.|
|Ejercicio4_HenaoLuis.psc|El programa es para ver si un numero es primo o no, solicita un número y da como resultado si el número es primo o no.|
|Ejercicio5_HenaoLuis.psc|El programa es para pasar de grados Celsius a Fahrenheit, solicita una temperatura en grados Celsius y da el resultado en grados Fahrenheit.|
|Ejercicio6_HenaoLuis.psc|El programa es para ver si un numero es par o impar, solícita ingresar un número y da como resultado si es par o no.|
|Ejercicio7_HenaoLuis.psc|El programa es para ver el area de un triángulo, solicita ingresar la base y la altura de un triángulo y da como resultado del área de ese triángulo.|
|Ejercicio8_HenaoLuis.psc|El programa es para ver la secuencia Fibonacci y pregunta cuantos números de la serie quiere ver y le muestra esa cantidad de números de la secuencia Fibonacci.|
|Ejercicio9_HenaoLuis.psc|Es un programa para ver la tabla de multiplicación, primero el número del que quiere ver su tabla de multiplicar, después pregunta cuantos números quiere ver de la tabla y le da como resultado la cantidad de números  de la tabla del número solicitado.|
|Ejercicio10_HenaoLuis.psc|El programa es para saber el promedio entre 10 números, solicita los 10 números y posteriormente da el promedio entre ellos.|
|Ejercicio11_HenaoLuis.psc|El programa es para saber el área de un círculo con su radio, solicita el radio y da como resultado el área del círculo.|
